# Historical Architecture Analyses

This folder contains one-off architecture analysis documents kept for provenance.

## Canonical Architecture Docs

Use these instead for the living architecture:

- `docs/architecture.md`
- `docs/architecture/`
- `docs/adr/`

## Why this folder still exists

These documents can still be useful for background context, conflict analysis, and understanding how the architecture narrative evolved over time.

However, they may contain:

- point-in-time counts
- superseded readiness language
- observations that were later narrowed or corrected

Do not treat this folder as the current source of truth for launch/readiness decisions.
