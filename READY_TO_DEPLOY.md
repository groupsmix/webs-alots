## 🎯 DEPLOYMENT STEPS

### 1. Install Dependencies (1 minute)
```bash
npm install
```

### 2. Build Application (3 minutes)
```bash
npm run build
```

### 3. Apply Database Migrations (2 minutes)
```bash
supabase db push
```

### 4. Set Environment Variables (5 minutes)
Add to `.env.production`:
```bash
# AI
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...

# WhatsApp
META_WHATSAPP_PHONE_NUMBER_ID=...
META_WHATSAPP_ACCESS_TOKEN=...

# SMS
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
TWILIO_PHONE_NUMBER=...

# Email
RESEND_API_KEY=...
RESEND_FROM_EMAIL=noreply@oltigo.com

# Cron
CRON_SECRET=...
```

### 5. Deploy (5 minutes)
```bash
npm run deploy
```

### 6. Test (10 minutes)
- Navigate to `/admin/ai`
- Test all 7 tabs
- Run analysis
- Approve an action
- Check notifications
